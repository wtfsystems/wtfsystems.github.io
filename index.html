WTEngine

A cross-platform C++17 game engine that uses Allegro and PhysicsFS. Has an ECS based memory manager, messaging system, dynamic maps for configuration variables, and a programmable menu interface. Performs 2D rendering and has simple controller support.

ppms

Python Polyphonic MIDI Synthesizer. Digital synthesizer playable with a MIDI input device. Has plug-in support for modules that alter the signal.
