Python/Flask Web App Framework

Python web app framework using Flask. Provides a login framework to base small web gui projects around.

Authenticates a user against a SQLite database. Passwords are hashed and stored using DES encryption.

Requirements

Requires the following packages to be installed:

  • flask
  • pyDes

Note: Make sure to configure https to avoid sending credentials in plain text!

User Database Admin

The script user_admin.py creates a useable database file for the framework. Also has commands to add, delete and list users.

Commands:

  • –new_db - Generates a new database file if one does not already exist.
  • –list - List users in the database.
  • -n username password - Adds a new user
  • -d username - Deletes a user

Configuration

All configuration is done in the framework_config.py file.

You should change the following settings and keep them private:

DES_KEY = b'\0\0\0\0\0\0\0\0'

Set the key used for DES encryption.

FLASK_SECRET_KEY = b'_5#y2L"F4Q8z\n\xec]/'

Set the secret key Flask uses to encrypt the session data.

FLASK_DEBUG_MODE = True

Toggle Flask debug mode on/off.
